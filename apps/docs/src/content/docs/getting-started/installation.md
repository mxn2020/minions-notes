---
title: Installation
description: Install Minions Notes packages
---

## TypeScript / Node.js

```bash
npm install @minions-notes/sdk minions-sdk
```

## Python

```bash
pip install minions-notes
```

## CLI

```bash
npm install -g @minions-notes/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
