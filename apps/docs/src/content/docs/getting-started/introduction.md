# Minions Notes

Freeform annotations and observations attachable to any Minion

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-notes/sdk`) — Core library for Node.js
- **Python SDK** (`minions-notes`) — Python bindings
- **CLI Tool** (`@minions-notes/cli`) — Command-line interface
- **Web App** — Interactive playground at [notes.minions.wtf](https://notes.minions.wtf)
